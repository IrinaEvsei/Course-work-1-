Результатом работы будет библиотека, которая позволит работать с картинками. 
Она будет представлять собой набор классов, которые позволят мне делать манипуляции с изображениями: 
для начала, конвертация из одного формата в другой; уменьшение ширины и длины картинки пропорционально по большему краю; поворот; черно-белый фильтр. 
Будет поддерживаться ограниченное расширение.
В начале это будет консольная утилита для этих действий. Параметры для действий будут задаваться с консоли.
В дальнейшем я буду видеть как продвигается работа с изменением картинок и, возможно, в демонстративных целях будет сделано десктопное приложение,
используя Java Swing. Я думаю, в процессе работы над курсовой будут расширены возможности данной библиотеки.
Методы и классы будут упакованы в JavaFile, которые будут доступны в любом другом Java-приложении.
